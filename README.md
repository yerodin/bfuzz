# Getting Started
  
  1. `git clone https://github.com/yerodin/bfuzz.git`
  2. `cd bfuzz`
  3. `cargo build`
  4. Binary located at `./target/debug/bfuzz`
  5. `cp ./target/debug/bfuzz $HOME/.local/bin` or whatever you prefer!
